# Personal Portfolio

Created with [`sv`](https://github.com/sveltejs/cli) v0.8.3.

```bash
bunx sv create
# Used prettier, eslint, tailwindcss, paraglide, bun, static-adapter, ... (TODO)
```

## Developing

```bash
bun run dev # --open

bun run format
bun run lint
bun run test
```

## Building

- Using the `static-adapter`.

```bash
bun run build
bun run preview # for the production build.
```
